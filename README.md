# 🍽️ Frontend - Sistema de Gestión de Servicio de Comida

⚠️ Este repositorio corresponde únicamente al **Frontend** del sistema.  
👉 Si deseas ver el repositorio del **Backend (PHP + MySQL)**, visita mi perfil y selecciona el repositorio correspondiente al **Backend**.

## 🛠 Tecnologías utilizadas

- React 18.2.0+
- JavaScript (ES6)
- Bootstrap (para estilos)
- Axios (para consumo de APIs)

## 📚 Descripción

Interfaz de usuario para el sistema de gestión de un servicio de comida. Este proyecto permite a clientes, encargados, cocineros y administradores interactuar con el sistema de manera intuitiva y en tiempo real.

## 📂 Funcionalidades del Frontend

- Página de inicio con menú accesible para cualquier usuario.
- Registro y login de usuarios.
- Paneles personalizados según tipo de usuario.
- Visualización de productos, combos y menús disponibles.
- Gestión visual y seguimiento de pedidos.
- Simulación de pagos (efectivo, tarjeta).
- Historial de pedidos por usuario.

## 🧑‍🍳 Tipo de negocio elegido

> Restaurante de especialidades
